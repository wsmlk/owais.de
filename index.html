Datenschutzerklärung

Diese Website wird über GitHub Pages bereitgestellt. Beim Aufruf der Website können durch GitHub technisch notwendige Verbindungsdaten (z. B. IP-Adresse, Datum und Uhrzeit des Zugriffs) verarbeitet werden.

Es werden keine Cookies gesetzt, keine Analyse-Tools eingesetzt und keine personenbezogenen Daten durch den Betreiber erhoben oder gespeichert.

Weitere Informationen zur Datenverarbeitung durch GitHub finden Sie unter: GitHub Privacy Statement

Wenn Sie uns per E-Mail kontaktieren, werden die von Ihnen übermittelten Daten (z. B. E-Mail-Adresse, Inhalt der Nachricht) zum Zweck der Bearbeitung Ihrer Anfrage verarbeitet und auf Servern der Zoho Corporation gespeichert. Eine Weitergabe der Daten erfolgt nicht ohne Ihre Einwilligung. Die Daten werden gelöscht, sobald sie für die Bearbeitung nicht mehr erforderlich sind, sofern keine gesetzlichen Aufbewahrungspflichten bestehen.

Zur Bereitstellung von Dateien nutzen wir eine Instanz von Nextcloud, die über eine eigene Subdomain erreichbar ist („share.owais.de“). Diese wird auf Servern des Hosting-Anbieters Hetzner Online GmbH betrieben. Beim Zugriff auf diese Subdomain werden durch den Hostinganbieter automatisch Informationen in sogenannten Server-Logfiles erhoben und gespeichert. Dies sind insbesondere: IP-Adresse Datuum und Uhrzeit der Anfrage aufgerufene Seite/Datei übertragene Datenmenge Browsertyp und -version Betriebssystem Die Verarbeitung dieser Daten erfolgt zur Sicherstellung eines störungsfreien Betriebs sowie zur Verbesserung unseres Angebots und zur Gewährleistung der IT-Sicherheit. Sofern Sie über die Nextcloud-Dateifreigabe Dateien abrufen oder hochladen, werden die von Ihnen bereitgestellten Daten ebenfalls auf den Servern von Hetzner gespeichert und verarbeitet. Mit dem Hosting-Anbieter wurde ein Vertrag zur Auftragsverarbeitung gemäß Art. 28 DSGVO abgeschlossen.
